## 一、官方适配范围（修正 SESSION_SUMMARY 的推测）

| 类别 | 车型 |
|------|------|
| ✅ 领克 | 23 款 01/03/05 全系、09 全系、24 款 01、25/26 款 03+ |
| ✅ 吉利 | **21–26 款星越 L 全部** |
| ❌ 暂不支持 | 2025 款领克 03（后期适配）+ 任何"新款"系统 |

**前置条件**：系统能装第三方 App + 用户有自主装 App 能力（系统未锁）。
**架构差异**（21-23 vs 24-26）只是 DockBar 控制路径不同，**不是适配 vs 不适配**。

## 二、权限模型（关键依赖）

小八不直接管权限，依赖三方 App 链：

| App | 用途 |
|-----|------|
| **应用管家** | 主入口。提供 ADB 开关 + 一键授权 + 快捷权限管理（通知使用权） |
| **设置 App**（领克 OSN / 吉利 NOS） | 部分车型用来给通知使用权 |
| **小八自带「桌面设置 → 权限管理」** | 系统连接已建立后一键全授 |

授权流程（v2.3.7 起）：
1. 临时开启无线 ADB（24/25 款星越 L 有单独教程）
2. 应用管家 → 应用管理 → 小八桌面 → 选权限 → 全允许（**悬浮窗即使显示"允许"也要重新点一遍**）
3. 进小八「桌面设置 → 系统与连接」确认 ADB 已连接 → 一键授权

**对 xylauncher 的启示**：要么 (a) 自带 ADB 引导流程；(b) 文档明确依赖应用管家。当前 xylauncher 是 b。

## 四、状态栏 13 项可显示信息

剩余里程、油量、车内温度、车速、转速、时间/日期、定位、WiFi、蓝牙、热点、媒体音量、PM2.5、外部温度

**交互**：
- 油量：点击在百分比 / 升数 之间切换
- 时间/日期：点击进**屏保模式**（自动移位防烧屏 + 显示农历）
- WiFi/蓝牙/热点：点击查看状态 + 跳转系统设置；热点**只能查看不能切换**
- 媒体音量：点击调节
- 每一项可在「桌面设置 → 状态栏设置」里单独开关

**xylauncher 缺**：PM2.5、热点状态、屏保模式。

## 六、DockBar 双层结构

### 主页 DockBar（9 键，仅主页显示）

```
[快捷] [车辆设置] [悬浮地图] [音乐] [应用抽屉] [空调开关] [温度] [空调功能区] [多功能编辑]
```

- **空调开关**：长按调起系统空调 App
- **温度调节**：S(Sync)/L(主驾)/R(副驾)，按住 2s 调出**放大温度块**（不同颜色区分温度区间，行车更安全）
- **空调功能区**：左右滑动有多页（AC/AUTO/风速/循环/座椅加热/通风/按摩/香氛/打开后备箱（**二次确认**））
- **多功能编辑键**（一键三套触发）：
  - 单击 → 加插件/快捷方式
  - 长按 → 空调功能区编辑 / 双区空调切换 / 底部栏透明度
  - 双击 → 空调一键场景（**3 种自定义模式**）

### 悬浮 DockBar（非主页自动出现）

```
[主页] [返回] [多任务] [通知] [Mini空调] [常用 App 槽] [编辑]
```

- 5 秒无操作 → 半隐藏
- 拖到屏边 → 收成长条
- v2.3.7 新增：**闲时自动贴边隐藏**（可关）
- **Mini 空调面板**：分页式，包含主要功能（非主页快调）
- 通知键：单击显示/隐藏系统状态栏，**长按**显示/隐藏系统 Dock
- 编辑器支持：通知/全局飞屏按键开关、面板大小、每个按键颜色、边框颜色、常用 App 槽位

→ xylauncher 当前 FloatingDockService 只有 48dp 圆点 + 5 按钮竖排，**分页 Mini 空调 + 闲时贴边**是清晰的复刻方向。

## 七、场景助手（小八核心差异化模块）

桌面设置 → 场景助手。这是 xylauncher 目前**完全没有的模块**。

| 子功能 | 行为 | 推断 API |
|--------|------|---------|
| 挂挡音效 | 自定义档位音效，**P 档最长 20s** | 监听 gear signal + 播音频文件 |
| **D 档自动打开 360 全景** | 挂入 D 档自动调起 | gear==D → Intent `com.ecarx.parking` |
| 开门音效 | 可选输出声道（通知/媒体）+ 音量 | 监听 door signal + AudioManager |
| 自启管理 | 自定义时间自启 App | AlarmManager + Intent |
| **车机互传** | 二维码 → 手机扫码 → 文件存 `Download/xiaoba-flies/` | 内置 HTTP server + QR 编码本地 IP/端口 |
| **剪贴板同步** | 二维码 → 手机提交文本 → 车机自动复制 | 同上 + ClipboardManager |
| 副驾屏常亮（星越 L） | **警告可能烧屏** | funcId `0x20284800` 已知（见 ECARX_API_NOTES） |
| HUD 配置 | 自定义歌词 / 红绿灯倒计时 / 转向灯 | 见下节 |
| 设备激活状态 | 显示剩余使用时长 / 激活码 / 车架号 | 激活码授权制 |
| 重启桌面 | — | 自杀 + 重拉 Activity |

**车机互传协议（推断）**：
- 车机起 HTTP server 监听某端口
- 二维码内容 = `http://<车机IP>:<port>/upload?session=<sessionId>`
- 手机扫码访问网页 → 上传文件 / 提交文本
- 车机端轮询或长连接拿到内容 → 落盘 / 写剪贴板
- "**点击二维码可刷新会话**" → 每次扫码用一次性 sessionId

→ xylauncher 完全没规划，**复刻成本中等**（一个 HTTP server + QR 库），收益高。

## 八、HUD 配置（关键技术细节）

支持自定义元素：歌词 / 红绿灯倒计时 / 转向灯 / 其它行车数据。

**关键约束**：红绿灯倒计时**依赖高德地图 8.5 及以上版本**。
→ 说明 HUD 渲染层**不是自己实现导航**，而是**从高德 hook / 监听**拿到数据再二次渲染。
→ xylauncher Phase 14 自己解析渲染没问题，但红绿灯这类数据源**必须接高德**。

教程在老赵抖音：https://www.douyin.com/user/self?modal_id=7593018106289704201

## 九、空调系统的差异化点

1. **温度调节放大块**：按住 2s 调出，按温度区间染色，**驾驶安全设计**
2. **双区空调**：长按多功能键切换（默认是单区+S/L/R 切换）
3. **空调功能区横向多页**：把高级功能（按摩/香氛/后备箱）藏在第二页
4. **一键场景**：双击多功能键调出 3 个自定义场景（小八只有 3 个，xylauncher 已有 6 个）
5. **后备箱打开二次确认**：避免误触

→ xylauncher 已经有 6 个一键场景（Phase 11），但**触发是从设置进**。**双击 DockBar 键直接调起场景**更符合驾驶动线。

## 十三、从用户行为反推的 API/Intent 线索

| 用户可见行为 | 推断的实现层 |
|-------------|-------------|
| 挂 D 档自动开 360 | 监听 gear signal + `Intent → com.ecarx.parking` |
| 副驾屏常亮 | funcId `0x20284800`，循环 100×1ms（ECARX_API_NOTES 已记录） |
| 双区空调切换 | API A 的 zone 字段（zone=1 主驾 / zone=4 副驾） |
| 系统主题同步（高德昼夜） | 监听系统主题变化 + 写高德 SharedPreference / 调高德 Intent |
| 设备激活/车架号 | ECARX VIN 读取（funcId 待挖） |
| 闲时自动贴边 | WindowManager + LayoutChange + 计时器 |
| 二维码车机互传 | 内置 HTTP server（Ktor / NanoHttpd），二维码编本地局域网地址 |
| HUD 红绿灯 | hook 高德 ContentProvider 或解析广播 |

## 十四、xylauncher 对照清单（可作 v0.2 路线图）

### 已对齐 ✅
- 14 大插件里有 10 个有对应实现或 UI
- 6 个一键场景已超过小八 3 个
- 副驾屏常亮 funcId 已知
- 座椅记忆 funcId 已知
- HUD 模式切换（4 模式）已知

### 关键差距 ⚠️
- **场景助手**整个模块缺失（挂挡音效 / 自启管理 / 车机互传 / 剪贴板）
- **悬浮 Dock**功能太单薄（缺 Mini 空调分页、闲时贴边、编辑器）
- **应用抽屉**功能太简（缺置顶、卸载、布局切换）
- **空调一键场景**触发方式不顺（缺 DockBar 双击入口）
- **状态栏**缺 PM2.5 / 热点 / 屏保模式
- **插件通用手势**（长按删 / 双击改 / 三击移）尚未统一

### 不打算复刻 ❌
- 趣味插件（木鱼 / 今天吃什么 / 今天去哪玩）— 自用没必要
- 壁纸商城 — 自用没必要做服务端
- 激活码授权 — 自用开源
- 跑马灯氛围灯 — 看后续兴趣

### 差异化补充（小八没有但 xylauncher 可做）
- AR-HUD 真投屏（小八只做悬浮窗模拟）
- DashCam 文件监视 + 上传（小八没做）
- 飞屏到副驾屏 / 多 Display 完整 UI
- 拖拽编排（小八的手势体系反而更"碎"）
